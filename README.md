## 打个广告
 该系统的所托管的maven库均基于作者的另一个项目[Neko-Maven](https://github.com/thtTNT/neko-maven) , 通过阿里云的FC,OSS,TableStore等服务快速搭建一个低成本的maven私服。成本低于5元/月。
